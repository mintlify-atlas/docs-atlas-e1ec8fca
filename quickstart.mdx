## Overview

This guide will walk you through creating your first healthcare plan quotation from start to finish. The entire process takes less than 5 minutes and requires no account creation.

<Info>
  As a SaaS product, Cotizador Online OSPADEP is designed to get you from landing page to meaningful results as quickly as possible.
</Info>

**Real form validation example:**

```javascript
// From script.js - Age validation logic
function determinarGrupoEtario(edad) {
    if (edad <= 25) return "0-25";
    if (edad <= 35) return "26-35";
    if (edad <= 54) return "36-54";
    if (edad <= 59) return "55-59";
    return "60+";
}
```

<Note>
  Age groups determine pricing tiers. The system automatically calculates the correct tier for each family member.
</Note>

### Price Calculation Example

Here's how the system calculates prices for an OMINT plan:

```javascript
// From script.js:1053-1108 - OMINT pricing calculation
function calcularPrecioFinalOMINT(planOMINT, composicionFamiliar, edadTitular, edadPareja) {
    let precioTotal = 0;
    
    // 1. Primary subscriber (titular)
    const grupoEtarioTitular = determinarGrupoEtario(edadTitular);
    precioTotal += planOMINT.preciosPorEdad[grupoEtarioTitular].adultoConyugue;
    
    // 2. Partner/spouse if applicable
    if (composicionFamiliar.tienePareja && edadPareja) {
        const grupoEtarioPareja = determinarGrupoEtario(edadPareja);
        precioTotal += planOMINT.preciosPorEdad[grupoEtarioPareja].adultoConyugue;
    }
    
    // 3. Children under 25
    if (children.length > 0) {
        precioTotal += planOMINT.preciosPorEdad[grupoEtarioTitular].hijo1Menor;
        if (children.length > 1) {
            precioTotal += planOMINT.preciosPorEdad[grupoEtarioTitular].hijo2MasMenores * (children.length - 1);
        }
    }
    
    return Math.round(precioTotal);
}
```

**Real Example:**

Family of 4 (Adult 35, Partner 33, Children ages 8 and 10) selecting OMINT Plan 6500:

1. **Titular (age 35)**: Group "26-35" → $227,182/month
2. **Partner (age 33)**: Group "26-35" → $227,182/month
3. **First child (age 8)**: → $138,785/month
4. **Second child (age 10)**: → $120,514/month

**Total**: $713,663/month

</Note>

  **Compare across providers**: Prices and coverage vary widely. The system shows all options so you can make an informed decision.

## Troubleshooting

<AccordionGroup>
  <Accordion title="I can't proceed to Step 2">
    Ensure you've clicked on one of the four plan type options. The "Continuar" button will only activate after selection.
  </Accordion>
  
  <Accordion title="No plans are showing">
    Check that all required form fields are filled. The system requires valid ages for all family members.
  </Accordion>
  
  <Accordion title="Prices seem incorrect">
    Prices are calculated based on age groups and family composition. Verify all ages are entered correctly. Contact your administrator if pricing data needs updating.
  </Accordion>
</AccordionGroup>
