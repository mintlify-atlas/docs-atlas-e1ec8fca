## What is Cotizador Online OSPADEP?

Cotizador Online OSPADEP is a comprehensive healthcare plan quotation system designed specifically for OSPADEP (Obra Social del Personal de Aeronavegación de Entes Privados) members. The platform enables users to compare and quote healthcare plans from multiple providers with real-time pricing and personalized recommendations.

<Note>
  Prices are updated monthly. Current version: **February 2026**
</Note>

## Key Capabilities

### For Members

- **Multi-step quotation flow**: Intuitive 3-step process from plan selection to final quotation
- **Multiple healthcare providers**: Compare plans from OMINT, Swiss Medical, Medicus, Galeno, Activa Salud, MEDIFE, and OSPADEP Salud
- **Personalized pricing**: Age-based calculations with family composition support
- **No account required**: Start quoting immediately without registration

### For Administrators

- **Dynamic pricing management**: Update prices by percentage or individually
- **Multi-user system**: Role-based access control (Super Admin, Admin, User)
- **Audit logging**: Complete activity tracking and session management
- **Price history**: Track all pricing changes with automatic versioning
- **External data management**: JSON-based pricing system for easy updates

### Pricing Structure Types

The system supports multiple pricing structures to accommodate different provider models:

| Structure Type | Description | Providers |
|----------------|-------------|----------|
| **estructura_compleja** | Age-based pricing with role differentiation (adult, spouse, children) | OMINT, Activa Salud, OSPADEP Salud |
| **plantilla_adultos_solo** | All family members priced as adults, age-based only | Swiss Medical |
| **plantilla_adultos_simple** | Simplified 2-tier age groups (≤65, >65) | SW Nubial, Swiss |
| **estructura_matrimonio_hijos** | Separate pricing for individual, spouse, and children categories | MEDIFE |

### Authentication System

The platform includes a comprehensive authentication system powered by `auth-system.js`:

```javascript
// Default credentials (located in auth-system.js:35-46)
username: 'ospadep_admin_2024'
password: 'OSPADEP_SecureAdmin_2024!'
role: 'super_admin'
```

**Session Management:**
- 8-hour session timeout with automatic extension on activity
- 30-minute inactivity auto-logout
- Session restoration on page reload
- Complete audit trail of all login/logout events

## Technical Specifications

- **Technology Stack**: Pure JavaScript (ES6+), HTML5, CSS3
- **Data Format**: JSON for external pricing data
- **Storage**: Browser localStorage for session and audit data
- **Architecture**: Client-side SPA (Single Page Application)
- **Price Updates**: Monthly via `precios-data.json` file replacement
- **Browser Support**: Modern browsers with localStorage support

<Warning>
  This is a client-side application. All data is stored locally in the browser. For production deployment, consider implementing server-side authentication and data persistence.
</Warning>
