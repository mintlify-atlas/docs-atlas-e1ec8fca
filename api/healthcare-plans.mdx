## Overview

Healthcare plans in OSPADEP are organized by provider with provider-specific pricing models and features.

---

## Plan Providers

### OMINT

**Pricing Model:** Role-based with age brackets

**Available Plans:**
- `plan2500`: Economic plan
- `plan4500`: Intermediate plan
- `plan6500`: Comprehensive plan (recommended)
- `plan8500`: Premium plan
- `planComunidad`: Community plan without copayments

**Age Brackets:**
- 0-25 years
- 26-35 years
- 36-54 years
- 55-59 years
- 60+ years

**Pricing Roles:**
```javascript
{
  adultoConyugue: number,    // Primary holder and spouse
  hijo1Menor: number,        // First child under 25
  hijo2MasMenores: number    // Additional children under 25
}
```

**Example Plan:**

```json
{
  "name": "PLAN 6500",
  "type": "omint",
  "preciosPorEdad": {
    "26-35": {
      "adultoConyugue": 227182,
      "hijo1Menor": 138785,
      "hijo2MasMenores": 120514
    }
  },
  "features": [
    "Cobertura integral OMINT",
    "Red de prestadores nivel 6500",
    "Internación en habitación privada",
    "Consultas médicas sin límite"
  ],
  "recommended": true
}
```

### Swiss Medical

**Pricing Model:** Simple age-based (all members as adults)

**Available Plans:**
- `po62`: Comprehensive coverage
- `po64`: Premium coverage (recommended)
- `sb04`: Basic coverage

**Age Brackets:**
- ≤65 years
- >65 years

**Pricing Structure:**
```javascript
{
  "≤65": 332704,
  ">65": 965409
}
```

**Key Features:**
- No discounts for children
- All family members charged as adults
- Only two age groups
- Premium provider network

## Plan Features

Common features across plans:

### Coverage Features
- Medical consultations (unlimited or limited)
- Emergency services 24/7
- Diagnostic studies (basic/intermediate/complex)
- Hospitalization (private room or shared)
- Dental coverage (complete/basic/none)
- Preventive medicine
- High complexity procedures

### Network Features
- Provider network level (2500, 4500, 6500, 8500 for OMINT)
- Geographic coverage (national, CABA/GBA)
- VIP/Priority access
- International coverage

## Recommended Flag

```javascript
"recommended": true
```

Plans marked as `recommended` are:
- Best value for money
- Most popular with clients
- Highlighted in the UI
- Typically mid-tier plans with comprehensive coverage

**Recommended Plans by Provider:**
- OMINT: Plan 6500
- Swiss Medical: PO64
- MEDIFE: Plata
- OSPADEP SALUD: OS300

## Plan Selection Logic

```javascript
// Get all plans from a provider
const omintPlans = prestadoresData.omint.planes;

// Filter recommended plans
const recommendedPlans = Object.entries(omintPlans)
  .filter(([key, plan]) => plan.recommended)
  .map(([key, plan]) => plan);

// Get plan by key
const plan6500 = omintPlans.plan6500;

// Check features
if (plan6500.features.includes('Internación en habitación privada')) {
  console.log('Private room included');
}
```

## Plan Comparison

```javascript
function comparePlans(plan1, plan2, age = 35) {
  const price1 = getPriceForAge(plan1, age);
  const price2 = getPriceForAge(plan2, age);
  
  return {
    plan1: { name: plan1.name, price: price1 },
    plan2: { name: plan2.name, price: price2 },
    difference: Math.abs(price1 - price2),
    cheaper: price1 < price2 ? plan1.name : plan2.name
  };
}
```
