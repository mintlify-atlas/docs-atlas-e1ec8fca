## Overview

Cotizador Online OSPADEP is a comprehensive healthcare plan quotation system designed for OSPADEP (Obra Social del Personal de Aeronavegación de Entes Privados). The platform provides an intuitive multi-step quotation flow for users to explore healthcare plans, combined with powerful administrative tools for managing pricing and monitoring system activity.

## Healthcare Providers

The system supports quotations for multiple healthcare providers:

- **OMINT** - Plans 2500, 4500, and 6500 with age-stratified pricing
- **SWISS MEDICAL** - Multiple plan tiers with family coverage options
- **MEDICUS** - Comprehensive healthcare plans with flexible pricing
- **GALENO** - Specialized medical coverage with various plan levels

<Note>
All pricing data is managed through an external JSON file for easy updates and maintenance.
</Note>
